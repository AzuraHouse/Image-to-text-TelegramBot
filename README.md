OCR Telegram Bot

Распознаёт текст на фото и документах с помощью Tesseract OCR.

Принимает фото и файлы (jpg/png)
Распознаёт русский + английский
Отправляет текст (или файл, если длинный)
Асинхронный с обработкой ошибок

Технологии
Python + aiogram 3.x
`pytesseract`, `Pillow`
`.env`, логирование

Как запустить
```bash
pip install -r requirements.txt
